# Explore Bali

Explore Bali is a web application that helps users explore tourist destinations in Bali with interactive maps, images, and multimedia content.

## Live Demo
https://explorebali.vercel.app/

## Features
- Explore popular tourist destinations in Bali
- Interactive map integration
- Image and multimedia content
- User-friendly web interface

## Tech Stack
- Next.js
- React
- Tailwind CSS
- JavaScript / TypeScript
